# Chess
A simple program to play Chess!

### Installing

Clone the repository & install TKinter
```
git clone https://github.com/Dapp3rDuck/pychess.git
py -m pip install tk
py main.py
```

### Built With

* [TKinter](https://docs.python.org/3/library/tkinter.html) - Python GUI
