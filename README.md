# Chess
A simple program to play Chess!

### Installing

Clone the repository & install PyGame
```
git clone https://github.com/Dapp3rDuck/pychess.git
py -m pip install tk
py main.py
```

### Built With

* [PyGame](https://pypi.org/project/pygame/) - Python GUI
